How to run the software?
1) Create a database using the code in dost.sql.
2) Copy paste the folders dost and dost_website in the root folder of the server.
3) Connect the laptop where the dost ,dost_website, database and the phone to the same network.
4) The database, dost and dost_website should be in the same laptop.
5) The ip of the api calls is hardcoded so that has to be changed according to the network ip.
